# sprite-ui-kit-generator

Generates deterministic pixel UI assets and validates required files/dimensions.

## Usage Examples

```text
Use $sprite-ui-kit-generator to build tileset, coin, question block, clouds, bitmap font atlas, and Bart avatar sprites.
```

```text
Use $sprite-ui-kit-generator and wire prebuild so all required PNGs regenerate before every build.
```
